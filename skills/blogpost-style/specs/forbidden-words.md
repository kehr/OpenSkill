# Forbidden Vocabulary

## Purpose

A shared blocklist of phrases that signal marketing-speak, untethered hyperbole, or AI-watermark leakage. Applied across all three style tracks. Matches inside fenced code blocks and inline code spans are ignored.

## How to Apply

1. Scan prose only. Skip ranges enclosed by ```` ```...``` ```` and `` `...` ``.
2. For each match, choose one of three actions:
   - **Replace with concrete content** (number, scenario, or specific example) when the original sentence carries a real claim.
   - **Delete** when the word is purely intensifier rhetoric.
   - **Rewrite the surrounding sentence** when the marketing word is load-bearing in the sentence's grammar.
3. Never substitute a synonym from the same forbidden category. Replacement must come from outside the blocklist.

## Categories

### Marketing metaphors

数字大脑, 数字躯干, 数字心智, 神经中枢, 灵魂, 心脏, 引擎心脏, 智慧之光, 智能跃迁

### Hyperbolic adjectives and adverbs (without numerical backing)

彻底, 全面, 极致, 深刻, 深度（修饰非技术名词时）, 强大, 卓越, 惊人, 震撼, 革命性的, 颠覆性的, 史诗级的, 现象级的（修饰一般产品时）

### Hyperbolic verbs

颠覆, 革新, 引爆, 崛起, 爆发（描述一般技术发展时）, 跃升, 蜕变, 重塑（描述非具体过程时）, 横空出世

### Industry grand-narrative phrases

范式革命, 范式转移, 全新时代, 行业里程碑, 生态变革, 百花齐放, 群雄逐鹿, 弯道超车, 改变世界, 重新定义{行业}

### Four-character cliche stacking (chengyu fillers)

水到渠成, 应运而生, 鳞次栉比, 不一而足, 有口皆碑, 一骑绝尘, 风起云涌, 蔚然成风, 大放异彩, 如火如荼, 遥遥领先

### Numeric-prefix headings (in section titles only)

第一章, 第二章, 第N章, 第一部分, 第N部分, Chapter 1, Chapter N, Part 1, Part N

### AI watermarks (anywhere in body, commit messages, branch names, file names)

AI 生成, AI-generated, generated by Claude, generated by AI, Co-Authored-By: Claude, Anthropic, OpenAI（提及组织名做技术引用时允许，作为生成水印时禁止）, Copilot 协作, ChatGPT 协作

### English marketing fillers

revolutionary (without measurement), groundbreaking, paradigm shift, game-changing, cutting-edge (in body prose), best-in-class, world-class, unparalleled, seamless (without technical specificity), unleash, supercharge

## Replacement Patterns

| Original (forbidden) | Replacement strategy |
|---|---|
| `X 显著提升了 Y` | `X 把 Y 从 A 提升到 B`（带数字） / `X 在 {场景} 下提升 Y，但在 {场景} 下不适用`（带 caveat） |
| `这是一场范式革命` | 删除整句，让事实自己说话 |
| `全面赋能开发者` | `让开发者在 {具体场景} 中节省 {具体动作}` |
| `数字大脑` | `语言模型` / `推理引擎` |
| `第三章：架构解构` | `Agent 的核心架构` 或 `Agent 是怎么搭起来的？` |
| `revolutionary breakthrough` | `improves X by N% on benchmark Y` |

## False-Positive Notes

- `深度学习`, `深度强化学习` are technical proper nouns and are not flagged.
- `卓越` inside a quoted user testimonial is allowed; flag it only when in author voice.
- `开放` and `生态` alone are fine; `生态变革` is the flagged compound.
- Code identifiers like `ground_breaking_feature_flag` inside backticks are ignored per the prose-only rule.
